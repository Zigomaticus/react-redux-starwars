Приложение на reacr + redux. Используются почти все основные хуки и возможности react: state, routing, context и т.д. С помощью redux можно добавлять/удалять из избранных, данные для этого хранятся в localstorage. Стилизация с помощью модулей css. 

Смотреть: https://zigomaticus.github.io/react-redux-starwars/
